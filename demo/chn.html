合并，操作如下：
$ git checkout master #
将当前分支切换为
master
$ git merge local #
将

当 一 个 分 支 检 查 无 误 并 且 与

注意 ，

分 支 是 无 法 删 除 的 。如 果 想 不 问 青 红 皂 白 地 删 除 一 个 分 支 ，可 以 使 用
git-branch
的
-D
选项。
